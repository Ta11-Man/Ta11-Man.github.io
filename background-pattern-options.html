Background Pattern Options for Your GitHub Page

1. Dot Grid (Subtle)
Clean and minimal dot pattern. Great for a professional look without being distracting.
background-image: radial-gradient( circle, rgba(7, 255, 237, 0.15) 1px, transparent 1px ); background-size: 30px 30px;
2. Grid Lines
Classic grid pattern. Very clean and organized, like graph paper.
background-image: linear-gradient( rgba(7, 255, 237, 0.1) 1px, transparent 1px ), linear-gradient(90deg, rgba(7, 255, 237, 0.1) 1px, transparent 1px ); background-size: 50px 50px;
3. Diagonal Stripes
Adds a sense of movement and energy. Modern and dynamic feel.
background-image: repeating-linear-gradient( 45deg, transparent, transparent 35px, rgba(7, 255, 237, 0.08) 35px, rgba(7, 255, 237, 0.08) 70px );

How to Apply

To add any of these patterns to your page, simply add the background CSS to your body style. For example:

body { margin: 0; overflow-x: hidden; overflow-y: auto; width: 100vw; min-height: 100vh; scroll-behavior: smooth; background-color: var(--bg-color); /* Add pattern here: */ background-image: radial-gradient( circle, rgba(7, 255, 237, 0.15) 1px, transparent 1px ); background-size: 30px 30px; text-align: center; }

Recommendation: I suggest starting with Pattern 1 (Dot Grid) or Pattern 2 (Grid Lines) as they're subtle and professional. Pattern 5 (Circuit Board) is great if you want a more tech-focused aesthetic.
